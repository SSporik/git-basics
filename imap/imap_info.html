Change Log

as of 2/28/2014
  • Added "Zoom to County" ability via zoom-to-feature tool
  • Added "Home Button" for user to return to map's initial extent
  • Added Query ability for a provided ArcGIS Online (AGOL) webmap via URL or "hard coded" within app.js
  • Added Query Tool and floating pane for Query Results
  • Added ability to download query results into comma separated value (.CSV) file from the Floating Pane
  • Ability to sort query results in floating pane
  • Added more comments in code
  • Added Measure Tools
  • Added Drawing Tools

Known Bugs

Item/Tool | Bug
  • Query | The "Circle" spatial select does not work in IE (version 10 tested)
  • Print | Print tool doesn't always work--any browser; either 404 or 500 error
  • Drawing, Measure | Having both the Drawing and Measure tools open at the same time may casue some interference with each other
  • Print | Print Tool doesn't show a user that the tool is working or "in process", esp for some webmaps that take longer than others
  • TOC, Legend | Some webmaps may not display legend graphics in the Table of Contents
  • Print | Print Tool button may jump place within toolset display during a interactive session
